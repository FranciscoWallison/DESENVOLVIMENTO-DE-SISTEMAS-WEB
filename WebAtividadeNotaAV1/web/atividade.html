Atividade Extra


Esta avaliação é composta de 3 Questões.
Cada questão certa corresponderá a 1 score. Totalizando o máximo de 3 scores.
Não existe atribuição de 0,5 score. O aluno deve acertar toda a questão para receber o score, caso contrário, será atribuído zero na questão.
Utilize o slides das aulas, exemplos das aulas como referência de consulta.

A nota final de AV1 será dada pelo cálculo:
AV1 = AV1 + (10 - AV1)/10*(Scores obtidos)

1ª Questão:

A aplicação inicia com uma página de login, na qual o usuário deve informar matrícula e senha.
Na definição do formulário, na página index.jsp, é prevista a utilização da função validaSenha()
( <form action="acesso.fic" method="post" onsubmit="return validaSenha();" > ).
O objetivo desta função é controlar a submissão do formulário, o qual só será submetido ao servidor se o retorno for "true"

A aplicação já está chamando corretamente a função, porém ela não está definida.
Defina e implemente a função validaSenha, a qual deve:
  • Caso a senha seja numérica
    - retornar o boolean true
  • Caso a senha não seja numérica
    - gerar um alerta com a informação: "A senha deve ser numérica!"; e
    - retornar o boolean false

Descubra qual é a senha correta
  • Consulte a Internet para "desvendar" o hash que você encontrou

Não esqueça de testar o código para verificar o cumprimento dos quesitos.


2ª Questão:

Após a autenticação, o usuário é redirecionado para uma página com alguns formulários.
Dentre os formulários há 1(um) que corresponde a uma consulta de alunos por nome.

Faça com que, na resposta do servidor ao clique do botão "Obter alunos por nome", além da tabela com a listagem dos alunos que correspondem a pesquisa (código já implementado) seja apresentado um alerta com a quantidade de alunos retornados na referida consulta

Não esqueça de testar o código para verificar o cumprimento dos quesitos.


3ª Questão:

O último button deve retornar uma listagem completa dos alunos.
Esta implementação não está funcionando.
O formulário na página chama a URL "lista.fic" que deve ser tratada no ServletControlador.
Para responder a esta requisição a aplicação deverá chamar o método getTodosAlunos() da classe DAO responsável pelas operações de banco de dados dos Alunos.
Após isso, deverá setar o retorno do método como atributo na requisição e encaminhá-la para o arquivo /jsp/listagemcompleta.jsp.
Estude e analise o código fonte para implementar corretamente esta funcionalidade.

O arquivo listagemcompleta.jsp não deve ser alterado.
A página pesquisa.jsp não deve ser alterada.
As únicas alterações devem ser feitas na classe ServletControlador.

Não esqueça de testar o código para verificar o cumprimento dos quesitos.
